# Page foot (portfolio)

The byline footer that closes every landing page — a ruled, muted sign-off with the
"built with" line. Links keep full ink so they read against the muted prose.

## Page foot
```html
<div class="page-foot">
  Part of <a href="/">tjakoen.github.io</a>, built with itself.
  <p>🤖 Built with Claude. I don't prompt and pray, I prompt and prove.</p>
</div>
```

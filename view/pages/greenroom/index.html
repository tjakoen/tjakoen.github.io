🎬 Greenroom

Run browser checks. Watch them pass. Hand over a bug bundle.

Greenroom is a small local app that runs automated browser checks against your web app and lets you review the results, with no coding required. It runs entirely on your own computer, from VS Code or the terminal. Under the hood it wraps Playwright in a friendly dashboard: point it at any site, tell it the URLs and how to log in, pick one or several environments, click Run, and watch a live pass and fail checklist. When something breaks, it hands a developer a one-click bug bundle with everything they need to reproduce it.

What it does

Run tests

Pick one or several environments, click Run, and watch a live pass, fail, and running checklist. More than one environment shows as a test-by-environment matrix with per-environment health.

Manage tests

Every check, grouped by feature. Run one, run a selection, or run by tag. Filter by failing, flaky, never-run, or slowest. Record a new test by clicking through the site, and let it log itself in without ever putting credentials in the file.

Report a bug

One-click bug bundle: the Playwright trace, video, screenshots, logs, and a plain-language summary, everything a developer needs to replay the session. Or export a single-frame PDF straight from the video.

Try the demo

Boots read-only with example data already filled in, so you can click around a populated dashboard with zero config before you set anything up.

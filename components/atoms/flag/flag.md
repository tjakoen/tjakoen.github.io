# Flag (portfolio)

A small status flag for a work — "Planned", "Demo" — an uppercase whisper in a dashed
pill. Portfolio-owned on purpose: inside GRAIN the dashed edge is the "AI / in-transit"
signal, so a static flag that borrows the look stays out of the design system.

## Flag
```html
<span class="flag">Planned · only the plan exists so far</span>
```
